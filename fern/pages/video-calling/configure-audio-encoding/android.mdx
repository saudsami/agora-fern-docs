## Implementation

| API | Description |
|:---|:---|
| `create(config.mAudioScenario)` | Set the audio scenario while creating an `RtcEngine` instance. |
| `setAudioProfile(profile)` | Set audio encoding properties before or after joining a channel. |
| `setAudioScenario` | Set an application scenario before or after joining a channel. |

### 1-on-1 interactive teaching

<CodeBlocks>
  <CodeBlock language="java" title="Java">
```java
  config.mAudioScenario = Constants.AudioScenario.getValue(Constants.AudioScenario.DEFAULT);
  engine = RtcEngine.create(config);
  RtcEngine.setAudioProfile(Constants.AUDIO_PROFILE_DEFAULT);
```
  </CodeBlock>
  <CodeBlock language="kotlin" title="Kotlin">
